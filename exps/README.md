# Experiment Short Routes

This folder contains short public routes for active experiment dashboards.

- `symbolic-tg/`: inspection dashboard for fine-grained temporal grounding in symbolic music generation and editing.
- `symbolic-captioning/`: versioned dashboard for symbolic music captioning pipelines, inputs, and Gemini results.

Unlike redirect-only folders, these pages may keep page-local CSS, JavaScript,
JSON data, and small demo assets when the purpose is fast research inspection.
