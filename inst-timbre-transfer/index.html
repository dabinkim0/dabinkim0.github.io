Method Description
Stable Audio Open Baseline method for target timbre fidelity.
SAO-ControlNet Implemented with Stable Audio Open and Diffusion Transformer (DiT)-based ControlNet for source structure preservation.
SmartControl An approach using Control Scale Predictors (CSPs), which adjusts the intermediate feature maps from the pretrained SAO-ControlNet.
AdaTransfer (Ours) Our proposed method using a Pitch-Loudness Balancer for source-target context-aware timbre transfer.
