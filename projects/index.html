Granular Synthesis-based Orchestral Sound Generator

Personal Project

This project explores how orchestral textures can be produced from a minimal sample set using granular synthesis. The system was implemented end-to-end in code and used to generate a range of orchestral textures, including drum rolls, string legatos, and pad-like timbres, from only two 3-second source samples, with custom delay and reverb processing applied in the final composition.

MIDI Controller: 쌍빛나리 (Ssangbitnari)

Developers: 김건호, 김다빈, 양승현.

A performance-oriented MIDI controller that non-musicians could use intuitively in live performance. Using an Arduino equipped with gyroscope and accelerometer sensors, the controller estimates orientation and rotational motion, transmits the resulting data to PureData, and maps these measurements to sound parameters. Flipping the device switches MIDI channels, while LED brightness provides immediate performance feedback.

Interactive Music Performance System: Future Mouth

Personal Project

A live performance tool that lets users control a synthesizer with mouth movement captured through a webcam. The system tracks mouth size to modulate a low-pass filter (LPF) and uses mouse position to shape panning and volume, while an on-screen virtual character mirrors the performer's mouth gestures in real time to make the interaction legible.
