Data Music Performance: "Manmulgwanggaek, Ritual Cheongol (만물관객, 리추얼 천골)"

Dabin Kim (Developer)

Designed a data sonification algorithm in Max/MSP that transforms the respiratory resonance frequency data of four dancers into sound in real-time. By utilizing I Ching trigrams to generate harmony and rhythm, the system musically expresses the dancers' physical states, creating clear, high-pitched sounds for high-resonance states and complex, low-pitched sounds for low-resonance states via an FM synthesizer.

Tools: Max/MSP

Orchestral Sound Generation with Granular Synthesis

Dabin Kim (Developer)

A personal project focused on creating orchestral music using SuperCollider. I implemented a Granular Synthesis system from code to generate diverse sound textures—such as drum rolls, string legatos, and pad sounds—using only two short audio samples ('Bass Drum' and 'Strings Staccato'). The piece was completed by applying custom delay and reverb effects.

Tools: SuperCollider

MIDI Controller: "Ssangbitnari (쌍빛나리)"

Dabin Kim (Developer)

A team project to create a performance-oriented MIDI controller designed for intuitive use by non-musicians. Using an Arduino and a gyro/accelerometer sensor, the controller detects orientation and rotation, sending this data to PureData to manipulate sound effects. Users can flip the controller to change MIDI channels and rotate it to adjust parameters, with real-time feedback provided by the changing brightness of an LED.

Tools: Arduino, PureData

Virtual Space Aleatoric Music: "Gravity Resonance"

Dabin Kim (Planner, Developer)

An interactive aleatoric music experience in a virtual space, inspired by John Cage's Silence. I developed the environment in Unity and implemented spatial audio using the Google Resonance Audio package. Participants navigate the space, triggering everyday sounds through random collisions with objects, which combine with the continuously changing spatial reverb to generate a unique ambient music piece.

Tools: Unity, Google Resonance Audio

Interactive Music Performance System: "Future Mouth"

Dabin Kim (Planner, Developer)

A live performance tool developed with P5.js that allows users to control a synthesizer with their mouth movements via a webcam. This interactive system tracks the user's mouth size to modulate a Low Pass Filter and uses the mouse position to control panning and volume. The experience is enhanced by an on-screen virtual character that mimics the user's mouth movements in real-time.

Tools: Javascript (P5.js)
