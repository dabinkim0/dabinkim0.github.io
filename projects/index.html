Data Music Performance: Manmulgwanggaek, Ritual Cheongol (만물관객, 리추얼 천골)

Designed a real-time bio-signal data sonification in Max/MSP that transforms the respiratory resonance frequency data of four dancers into sound.

Employing I Ching trigrams as a structural framework for harmony and rhythm, the system mapped performers' bodily states to musical behavior, generating clear, high-register tones for high-resonance states and denser, lower-register textures for low-resonance states through FM synthesis.

Orchestral Sound Generation with Granular Synthesis

A personal project exploring how orchestral textures can be produced from a minimal sample set using granular synthesis.

The system was implemented end-to-end in code and used to generate a range of orchestral textures, including drum rolls, string legatos, and pad-like timbres, from only two short source samples, with custom delay and reverb processing applied in the final composition.

MIDI Controller: Ssangbitnari (쌍빛나리)

A performance-oriented MIDI controller that non-musicians could use intuitively in live interaction.

Using an Arduino equipped with gyroscope and accelerometer sensors, the controller estimates orientation and rotational motion, transmits the resulting data to PureData, and maps these measurements to sound parameters. Flipping the device switches MIDI channels, while LED brightness provides immediate performance feedback.

Role

Tools

Virtual Space Aleatoric Music: Gravity Resonance

An interactive aleatoric music experience in virtual space.

I designed the environment in Unity and implemented spatial audio with the Google Resonance Audio package. Participants navigate the space and trigger everyday sounds through unpredictable collisions, which combine with evolving reverb behavior to produce a continuously changing ambient composition.

Role

Planner, developer

Tools

Interactive Music Performance System: Future Mouth

A live performance tool that lets users control a synthesizer with mouth movement captured through a webcam.

The system tracks mouth size to modulate a low-pass filter and uses mouse position to shape panning and volume, while an on-screen virtual character mirrors the performer's mouth gestures in real time to make the interaction legible as both instrument and performance.
