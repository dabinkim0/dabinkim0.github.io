# Publications Folder

Canonical publication demo pages use one folder per paper:

- `adatt/`: AdaTT demo, media, and spectrogram generation script.
- `ddsp-carsound/`: DDSP car-sound demo, media, and build script.
- `figs/`: representative figures for publications that link to external pages instead of local demo pages.

Compatibility-only routes:

- `timbre-transfer/`: legacy redirect to `adatt/`.

Do not place media or implementation files in compatibility-only folders. If an old public URL needs to keep working, add only a minimal `index.html` redirect and document the canonical target here.
